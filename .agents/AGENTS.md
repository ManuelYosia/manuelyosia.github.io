## Project Overview
This is a Next.js portfolio website for a software engineer. It is a static site that is deployed to GitHub Pages. The website is built with TypeScript, Tailwind CSS, and Next.js.

## Key Commands
- `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm run start`: Starts the production server.
- `npm run lint`: Runs the linter.

## Important Caveats
- The project uses TypeScript, which is a statically typed superset of JavaScript.
- The project uses Tailwind CSS, which is a utility-first CSS framework.
- The project uses Next.js, which is a React framework for server-side rendering and static site generation.
- The project is deployed to GitHub Pages.

## Folder Structure
- `pages/`: Contains the pages of the application.
- `components/`: Contains the components of the application.
- `data/`: Contains the data of the application.
- `styles/`: Contains the styles of the application.
- `.github/workflows/`: Contains the GitHub Actions workflows.
- `.next/`: Contains the Next.js build output.
- `node_modules/`: Contains the node modules.
- `package-lock.json`: Contains the package lock file.
- `package.json`: Contains the package file.
- `postcss.config.mjs`: Contains the PostCSS configuration.
- `README.md`: Contains the README file.
- `tailwind.config.ts`: Contains the Tailwind CSS configuration.
- `tsconfig.json`: Contains the TypeScript configuration.
- `.agents/`: Contains the agents configuration.

## Path
- `.agents/docs/personal-information.md`: Contains the personal information of the user.
